Prom.vim
========

Syntax and highlight for [Prometheus](https://prometheus.io) plain text [exposition format](https://prometheus.io/docs/instrumenting/exposition_formats/).

## Installation

With Packer.nvim:

```lua
use('belltoy/prom.vim')
```

With vim-plug:

```vim
Plug 'belltoy/prom.vim'
```
